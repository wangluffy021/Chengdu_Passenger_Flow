> 这是一个成都地铁客流量的可视化项目

#### 文件结构
Chengdu_Passenger_Flow/
├── 成都地铁客流量数据.xlsx    // 客流量数据
├── chengdu_metro_viz.html    // 关键代码及可视化页面
└── README.md                         // 说明

#### 需求：
- 成都地铁客流量数据
- vscode + live server插件（快速浏览html，实时更新）

#### 运行步骤
1. 选中文件chengdu_metro_viz.html ,右键选择"open with live server"
   在该项目中也可直接文件夹双击html文件打开。
2. 上传文件“成都地铁客流量数据.xlsx”。
